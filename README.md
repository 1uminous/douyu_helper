## 工具简介
利用Github Action的方式实现斗鱼TV自动获取粉丝荧光棒，并完成赠送工具，简单配置即可实现赠送每日赠送荧光棒

**如果认为工具还算好用，请顺手点一个Star吧~**

## 功能列表
* [x] 每天上午9点启动
* [x] 自动获取荧光棒
* [x] 自定义赠送的房间
* [x] 平均分配荧光棒至拥有粉丝牌的房间
* [x] 推送日志
